Basic Overview

Cells = power and heat

Power = money

Money = Bigger cells and more money

Lots of money = particle accelerators

Particle accelerators = exotic particles (EP)

EP = more upgrades and new toys for more power, heat and money!

TF = Time Flux = Offline time accumulated (speeds game up when enabled)

Mobile Macro Mode

On mobile, enable Macro Mode on the reactor screen to use bulk editing gestures:

  • Select a part, turn on Macro Mode, choose an action, then:
  • Paint — drag across tiles to place a line of parts
  • Row / Col / Check / Replace / Sell — tap a tile with that action selected
  • Two-finger tap — sell all matching parts on the tapped tile
  • Stride — spacing for row/column fills (same as desktop number keys 2–9)

Desktop hotkeys: Ctrl = row, Alt = column, Shift = ignore part filter, Ctrl+Alt+Shift = checkerboard.
